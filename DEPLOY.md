# 部署说明

这个工具有两种使用方式。

## 方式一：本地办展先用

适合你这次先试用。

1. 在电脑或平板上安装 Python。
2. 双击 `启动摆摊工具.bat`。
3. 后台打开 `http://127.0.0.1:8765/admin`。
4. 买家如果和你在同一个 Wi-Fi 下，可以打开 `http://你的电脑IP:8765`。

这个方式不需要服务器，但买家离开现场网络后就访问不到。

## 方式二：正式上线

适合买家离开队伍、走到场外也能继续打开订单页面。

你需要一个能运行 Python 或 Docker 的云服务器/云平台。部署时要注意一件事：

```text
data/booth.sqlite3 必须持久保存
```

如果平台重启后会清空文件，就要把环境变量设成：

```text
BOOTH_DATA_DIR=/data
```

并给 `/data` 配一个持久磁盘。

常见配置：

- 启动命令：`python app.py`
- 端口：读取环境变量 `PORT`
- 数据目录：`BOOTH_DATA_DIR=/data`

如果平台支持 Docker，也可以直接用项目里的 `Dockerfile`。

## 上线后的二维码

上线后会有一个网址，例如：

```text
https://你的摊位工具地址
```

把这个网址生成二维码贴在摊位上。后台地址是在后面加：

```text
/admin
```

第一次上线后请立刻进入后台修改默认密码。
