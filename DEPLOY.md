# 部署说明

这个工具有两种使用方式。

## 方式一：本地办展先用

适合先在自己的电脑或平板上试用。

1. 安装 Python。
2. 双击 `启动摆摊工具.bat`。
3. 后台打开 `http://127.0.0.1:8765/admin`。
4. 买家如果和你在同一个 Wi-Fi 下，可以打开 `http://你的电脑IP:8765`。

这种方式不需要服务器，但买家离开现场网络后就访问不到。

## 方式二：正式上线

适合买家离开队伍、走到场外也能继续打开点单页面。

你需要一个能运行 Python 或 Docker 的云服务器/云平台。部署时要注意一件事：

```text
data/booth.sqlite3 必须持久保存
```

如果平台重启后会清空文件，就要把环境变量设成：

```text
BOOTH_DATA_DIR=/data
```

并给 `/data` 配一个持久磁盘。

常见配置：

- 启动命令：`python app.py`
- 端口：读取环境变量 `PORT`
- 数据目录：`BOOTH_DATA_DIR=/data`

如果平台支持 Docker，也可以直接使用项目里的 `Dockerfile`。

## 后台地址

默认后台地址是在网站后面加：

```text
/admin
```

例如：

```text
https://你的网站地址/admin
```

如果你要正式办展，或者把项目开源给别人使用，建议设置环境变量 `ADMIN_PATH`，改成只有自己知道的后台路径：

```text
ADMIN_PATH=/your-private-admin-path
```

设置后后台地址会变成：

```text
https://你的网站地址/your-private-admin-path
```

买家二维码只放买家首页地址，不要公开后台地址。

## 上线后的二维码

上线后会有一个网站地址，例如：

```text
https://你的摊位工具地址
```

把这个买家首页地址生成二维码贴在摊位上。

## 初次上线

第一次上线后请立刻进入后台：

1. 修改默认后台密码。
2. 上传摊位 logo、欢迎语和收款码。
3. 设置摊员名单。
4. 确认商品、库存、订单流程都正常。

## 忘记密码

请查看 `PASSWORD_RESET.md` 里的《重置密码说明》。

重点是：密码保存在运行网站的数据库里，不保存在 GitHub 代码里。忘记线上密码时，需要在运行网站的服务器或云平台里执行重置脚本。
