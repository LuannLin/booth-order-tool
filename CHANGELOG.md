# 更新日志

本文件记录摆摊点单工具的重要功能新增、体验优化和问题修复。

## v1.1.3 - 2026-07-21

### 新增

- 商品管理支持全选当前搜索或筛选结果，并可批量上架、批量下架或批量删除。

### 优化

- 重新整理商品编辑表单，将价格、库存、低库存提醒、赠品状态和保存操作分区展示。
- 缩短销售日期选择栏，并重新排列日期范围与导出操作，在手机和桌面上更紧凑清楚。

### 修复

- 后台登录会话改为保存在数据库中，刷新页面或 Render 重启后不会立即退出登录。
- 后台登录会保留 7 天，并从服务器恢复当前摊员身份；主动退出后会立即失效。
- 后台设置接口不再向浏览器返回后台密码。

## v1.1.2 - 2026-07-20

### 优化

- 商品、Logo 和收款码图片改为独立加载并长期缓存，商品与摊位接口不再把图片原文塞进 JSON。
- 买家端只读请求支持网络波动自动重试，摊位设置和商品信息改为顺序读取。
- 后台新增“压缩现有图片”，可一次性把过大的商品图压缩为网页展示尺寸。
- 后续上传超过 1MB 的图片会在浏览器中自动压缩，减少保存耗时和服务器流量。

### 修复

- 将服务器连接等待队列从默认的 5 提升到 128，减少多人同时打开页面时的连接拒绝。
- 前后台只读请求及可安全重试的下单请求遇到 `429/502/503/504` 时会退避后重试。
- 修复前后台偶发 `Unexpected end of JSON input`、空响应和页面加载失败的问题。
- 下单请求增加唯一识别码；网络中断重试时会返回原订单，不会重复扣库存或生成重复订单。
- 订单创建成功后优先展示取单码，后续库存刷新失败也不会再遮住取单凭证。
- 编辑商品、复制新增商品或保存摊位设置时，新的图片地址不会覆盖数据库中的原图。

## v1.1.1 - 2026-07-20

### 修复

- 修复保存商品或摊位设置已经生效，但后台因服务器返回内容不完整而错误弹出 JSON 报错的问题。
- 保存接口改用轻量成功响应，不再重复回传商品图、Logo 和收款码图片，降低会场网络下的传输量。
- 修复网络波动时摊员名单读取失败，导致登录页无法选择正确摊员的问题。
- 后台定时刷新改为只读取订单，避免每 5 秒重复下载全部商品图和收款码；只读请求在响应中断时会自动重试。

## v1.1.0 - 2026-07-20

### 新增

- 商品管理支持按名称、作者、分类和标签搜索。
- 商品管理支持按在售、低库存、售罄、赠品、已下架和作者筛选。
- 商品管理支持多选商品并批量删除。
- 商品卡片增加“复制新增”，可复用原商品的价格、库存、分类、作者、标签、图片和状态，再修改信息后保存为新商品。
- 商品编辑表单增加作者候选项和常用标签快捷选择。
- 销售情况默认展示当天订单，并可切换日期或查看全部历史。
- 全部历史订单按照日期分组展示。
- 销售概览会根据当前筛选范围统计订单数、售出件数和营业总额。
- 增加作者分账统计，展示各作者的订单数、售出件数、赠品件数、原价销售额、分摊优惠和分账金额。
- 增加作者分账 CSV 导出功能。

### 优化

- 商品作者信息和标签在后台商品卡片中更加醒目。
- 商品图片上传区域增加实体图和 1MB 以内压缩提示。
- 将“商品销售排行”调整为“单制品销量统计”，更准确地表达数据含义。
- 重新整理销售概览、作者分账、单制品销量统计和订单明细的信息层级。
- 调整商品管理与销售情况页面的边距、字号、颜色、卡片尺寸和数据间距。
- 优化商品表单、筛选工具、销售统计和订单明细在手机、平板及桌面设备上的布局。
- 手机端长小票改为正文独立滚动，“我已截图”按钮始终保留在底部可点击位置。

### 修复

- 删除作者分账导出表中容易产生误解的“总计”行。
- 修复长取单清单在部分手机尺寸下遮挡“我已截图”按钮的问题。
