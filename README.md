# 漫展摆摊点单工具

一个给漫展、同人展、快闪摊位使用的网页点单工具。

买家扫码打开点单页，选择制品并生成取单码；摊主在后台查看订单、分配拣货、核验付款、记录取单和导出销售数据。工具默认不接入自动支付接口，适合现场使用微信/支付宝收款码或现金，并由摊主在取单前人工核验付款。

本项目由 **Luann** 和 **Vice Holloway** 共同开发完成，免费公开源码并允许非商用使用。欢迎大家基于这个版本自由修改、自用、二创和适配自己的摊位流程；如果公开分享改版或使用心得，可以注明来源或小红书 @ Luann /抖音 @JUPOO MIO。

为了避免泄露个人信息，本仓库不内置任何真实摊位网址、后台地址、社交平台账号、收款码或订单数据。

## 适合的场景

- 漫展现场排队点单，减少口头确认和反复算账。
- 买家暂时离开队伍，稍后凭取单码回摊位取货。
- 合摊或多人看摊，后台可以查看订单状态和拣货分配。
- 展后导出销售记录，核对商品销量、营业额和合摊分账。

## 主要功能

买家页：

- 展示摊位名、欢迎语、摊位 Logo。
- 展示商品名称、价格、图片、库存状态。
- 支持购物车、数量增减、库存不足提示和售罄沉底。
- 支持现在领取和稍后领取。
- 现在领取填写手机尾号后四位，稍后领取填写完整 11 位手机号和预计领取时间。
- 支持微信、支付宝、现金三种支付方式。
- 下单后生成取单码和小票样式清单，显示商品、赠品、优惠和应付金额。

摊主后台：

- 登录后管理商品、图片、价格、库存、标签和赠品属性。
- 编辑摊位名、欢迎语、Logo、微信收款码、支付宝收款码。
- 添加、修改和删除摊员，用于多人协作拣货。
- 查看今日订单，按新订单、拣货中、待取单、已完成分组处理。
- 支持拣货领取、释放拣货、转交拣货、逐项勾选拣货完成。
- 支持核验付款、标记已取单、撤回订单状态。
- 销售情况页可查看历史订单、按日期筛选、统计营业额。
- 支持导出订单明细和商品汇总表。

促销与赠品：

- 支持满额赠品。
- 支持买件赠品。
- 支持满额减价。
- 赠品会进入拣货清单和销售统计。
- 赠品库存不足时后台会提示，前台不会直接向买家暴露赠品缺货原因。

## 本地试用

1. 安装 Python 3.10 或更新版本。
2. 进入项目文件夹。
3. 双击 `启动摆摊工具.bat`。
4. 买家页打开：

```text
http://127.0.0.1:8765
```

5. 后台默认打开：

```text
http://127.0.0.1:8765/admin
```

6. 默认后台密码：

```text
123456
```

第一次进入后台后，请立刻到【摊位】里修改后台密码，并上传自己的 Logo 和收款码。

## 手机和平板测试

如果手机、平板和电脑在同一个 Wi-Fi 下，可以用电脑的局域网地址访问。

Windows 查看局域网地址：

```powershell
ipconfig
```

找到类似 `192.168.x.x` 的 IPv4 地址后，手机或平板打开：

```text
http://你的IPv4地址:8765
```

后台地址默认是：

```text
http://你的IPv4地址:8765/admin
```

如果已经设置了自定义后台路径，请用你自己的后台路径替换 `/admin`。

## 数据与隐私

运行数据默认保存在：

```text
data/booth.sqlite3
```

这里可能包含商品、订单、手机号、摊位设置、收款码和后台密码。开源或发给别人前，请确认不要上传 `data/` 文件夹。

本项目的 `.gitignore` 已经默认忽略：

```text
data/
*.sqlite3
*.db
.env
```

如果你使用 GitHub 网页手动上传文件，也请不要把 `data/` 文件夹拖进去。

## 后台安全

默认后台路径是：

```text
/admin
```

开源后，大家都会知道这个默认路径。正式办展时，建议在服务器或云平台设置环境变量：

```text
ADMIN_PATH=/换成只有你知道的后台路径
```

例如设置为 `/my-booth-workroom-2026` 后，后台地址会变成：

```text
https://你的网站地址/my-booth-workroom-2026
```

买家二维码只放买家首页地址，不要公开后台地址。

## 部署上线

正式办展时，建议部署到支持 Python 或 Docker 的云平台/服务器，并开启 HTTPS。部署后，把买家首页网址生成二维码贴在摊位上即可。

详细步骤见：

```text
DEPLOY.md
```

忘记后台密码时，见：

```text
PASSWORD_RESET.md
```
