# 漫展摆摊点单工具

这是一个先给漫展现场自用的网页小工具：

- 买家页：浏览商品、加入购物车、选择现在领取或稍后领取、生成取单码。
- 摊主后台：管理商品、编辑摊位信息、查看订单看板、核验支付、标记已取单、导出订单明细和商品汇总。
- 支付方式：微信、支付宝、现金。第一版不接自动支付，取货时出示付款成功页面，摊主人工核验。

领取信息规则：

- 现在领取：填写手机号后四位，取货时核对，避免拿错单。
- 稍后领取：填写完整 11 位手机号，并选择预计领取时间。

后台订单按钮会按状态显示：

- 新订单：只显示 `待拣货`
- 拣货中：显示 `待取单` 和撤回新订单
- 待取单：显示付款核验、`已取单` 和撤回拣货中
- 已完成：只保留撤回待取单

后台订单看板默认只显示当天订单，方便现场处理取单。

历史订单和营业额放在 `销售情况`：

- 可以看全部历史销售情况。
- 可以按日期筛选订单和营业额。
- 可以查看订单数、售出件数、营业总额和商品销售排行。

## 本地启动

1. 先安装 Python 3.10 或更新版本。
2. 双击 `启动摆摊工具.bat`。
3. 买家页打开：`http://127.0.0.1:8765`
4. 后台打开：`http://127.0.0.1:8765/admin`
5. 默认后台密码：`123456`

第一次进入后台后，建议先到“摊位”里改密码、上传 Logo 和收款码。

## 手机和平板一起试用

如果手机和平板在同一个 Wi-Fi 下，可以把电脑的局域网地址发给手机。

在 Windows 里查看本机地址：

```powershell
ipconfig
```

找到类似 `192.168.x.x` 的 IPv4 地址，然后手机打开：

```text
http://你的IPv4地址:8765
```

后台地址是：

```text
http://你的IPv4地址:8765/admin
```

## 数据位置

订单、商品和设置会保存在：

```text
data/booth.sqlite3
```

导出的表格在后台“订单”页：

- `导出订单明细`：逐笔订单和商品明细，方便核对每张单。
- `导出商品汇总`：按商品统计销量和营业额，方便清点销售额和合摊分钱。

## 部署建议

正式给买家扫码使用时，建议部署到支持 Python 的云平台或服务器，并开启 HTTPS。上线后再把买家页网址生成二维码贴在摊位上。

第一版适合用轻量云服务器、Render、Railway 或 Fly.io 这类平台。部署前需要确认平台能长期保存 `data/booth.sqlite3`，否则重启后数据可能丢失。
